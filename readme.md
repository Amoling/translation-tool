#### 使用方法：

- 在命令行中输入：
```
$ npm install translation-tool -g
```
安装本工具；

- 输入命令：
```
$ fanyi <word[phrase][sentence]>
```
或
```
$ translate <word[phrase][sentence]>
```
并回车，即可得到翻译结果。
